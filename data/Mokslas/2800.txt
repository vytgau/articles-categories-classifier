Gegužės 13 d. KTU Santakos slėnyje vyks Python programavimo kalbos konferencija – PyConLT'17, rašoma pranešime spaudai. Skelbiama, kad konferencijos programa paruošta taip, kad kitų kalbų naudotojai ten jaustųsi kaip namie, kad pranešimai būtų suprantami kitų kalbų naudotojams, taip pat vyks praktiniai užsiėmimai, kurie skirti darbo su Python patirties neturintiems dalyviams. Vyks svetainių kūrimo, duomenų mokslo ir Python pradmenų praktinės sesijos. Python programavimo kalba, TIOBE indekso duomenimis, pagal populiarumą užima ketvirtą vietą, po JAVA, C ir C++. Esminiai skirtumai tarp jų apibūdinami taip: Python yra intuityvi, lengvai naudojama, kokybiška kalba su daugybe įvairioms sritims skirtų bibliotekų, Java - naudojama versle, bankuose ir telefonuose, t.y. tokiose vietose, kur negali rinktis programavimo kalbos, todėl su ja programuojama ne iš noro, o iš pasirinkimo neturėjimo. O C arba C++ - naudojama dėl greičio ir labai gero suderinamumo su viskuo, tačiau C/C++ programavimas yra gana sudėtingas. Konferencijos organizatoriai skelbia, kad nuo 2014 m. Python - populiariausia programavimo įvado kalba JAV TOP universitetuose, o kasmetinės „StackOverflow“ apklausos respondentai ją pripažino geidžiamiausia išmokti programavimo kalba. Viena svarbiausių Python ypatybių - paprastumas. Loginį veiksmų grupavimą lemia atitraukimas nuo krašto, naudojama mažai skirtukų, nereikia iš anksto apsirašyti kintamųjų tipo. Todėl kodas lengviau suprantamas. Kita Python stiprybė - atviro kodo bibliotektų įvairovė. Github, BitBucket ir kitose talpyklose rasite svetainių, darbastalio programų ir žaidimų kūrimui, duomenų analizei, kriptografijai, moksliniams tyrimams bei turbūt visoms kitoms sritims skirtas bibliotekas. Paskutiniu metu Python populiarėjo duomenų mokslo ir automatinio mokymosi srityse. Vienas geriausiai Python potencialą srityje iliustruoja tai, kad neseniai „Google“ įsigytame „Kaggle“ duomenų mokslo konkurse Python populiarumu lenkia kitas kalbas. Python buvo populiariausias įrankis ir Vilniuje vykusiame dirbtinio intelekto hakatone „AI Camp”. Python kalba buvo naudojama kuriant „Google“, „YouTube“, „Quora“, „Disqus“, „Instagram“, „Pinterest“, „DropBox“ ir kitose svetainėse.
